# Feature: Landingpage

## Ziel
Eine **Sahne-Landingpage** die professionell überzeugt und den ersten Eindruck perfekt macht.

## Design-Prinzipien
- **Clean & Minimalistisch** – Weniger ist mehr
- **Restaurant im Fokus** – Fleischqualität hervorheben
- **Kühe NICHT prominent** – USP auf Unterseite verkaufen
- **Kurze Sektionen** – Tiefergehende Infos auf Unterseiten

## Sektionen

### 1. Hero
- **Fullscreen** mit hochwertigem Food/Restaurant-Bild
- **Headline:** Fokus auf Genuss & Qualität
- **Subline:** Kurz, emotional, einladend
- **CTAs:** "Restaurant entdecken" + "Tisch reservieren"
- **Keine Stats** – cleaner Look

### 2. Highlights (NEU)
- **3-4 Cards** in einer Reihe
- **Icons + kurze Texte**
- Themen: Restaurant, Fleischqualität, Historischer Hof, Reitbetrieb
- **Hover-Effekte** mit Framer Motion

### 3. Restaurant-Teaser
- Kompakte Version der RestaurantSection
- Menü-Highlights (2-3 Gerichte)
- CTA zur Speisekarte

### 4. Fleisch-Teaser
- "Premium Fleisch direkt vom Hof"
- Kurzer Text + CTA zur Fleischverkauf-Seite

### 5. Kontakt-CTA
- Einfacher Call-to-Action Block
- Telefon + Reservierung

## Technische Umsetzung
- React + TypeScript
- Framer Motion für Animationen
- Tailwind CSS v4 für Styling
- Lazy Loading für Bilder
